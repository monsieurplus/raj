Archive from 2007 : Java RPG engine made with default UI Components
Development has stopped for a long time, this is just archival deposit.

Archive de 2007 : Moteur de RPG en Java fonctionnant avec les éléments d'UI par défaut
Développement arrêté depuis longtemps, je dépose les fichiers ici pour mes archives personnelles.
Ce moteur a été réalisé en projet tutoré de 2ème année d'IUT informatique.

Spécifications
--------------
* Données de jeux (niveaux, obstacles, événements) sauvegardées en XML
* Détection de ollisions du personnage avec le décor
* Gestion d'événements codés en XML
* Sprites animés grace à la magie du gif animé
* Mode développeur permettant de gérer des sauvegardes, accéder aux variables du jeu et débugger le XML
* Deux mini-jeux de test fournis
